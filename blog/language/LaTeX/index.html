    LaTeX

    第一步：设置文件类型

    文件类型一般在源文件的第一行设置，文件类型十分重要，决定了这个文件的编写方式

    \documentclass{ctexart}

    语句一般也可以设置参数，如：

    \documentclass[12pt, a4paper, oneside]{ctexart}

    m默认字体大小为12pt，纸张大小为A4，单面打印

    正文

    正文部分需要放入document环境中：

    \documentclass[12pt, a4paper, oneside]{ctexart}

\begin{document}

这里是正文. 

\end{document}

    宏包

    为了完成一些功能，需要在导言区（也就是document环境之前）加载宏包。

    \usepackage{amsmath, amsthm, amssymb, graphicx}

    与数学相关的宏包：amsmath、amsthm、amssymb

    用于插入图片的宏包：graphicx

    另外，这条语句也可以设置参数：

    \usepackage[bookmarks=true, colorlinks, citecolor=blue, linkcolor=black]{hyperref}

    超链接宏包hyperref，设置引用颜色为黑色

    标题、作者、日期

    标题、日期、作者都需要放在导言区。

    \documentclass[12pt, a4paper, oneside]{ctexart}
\usepackage{amsmath, amsthm, amssymb, graphicx}
\usepackage[bookmarks=true, colorlinks, citecolor=blue, linkcolor=black]{hyperref}

% 导言区

\title{我的第一个\LaTeX 文档}
\author{Dylaaan}
\date{\today}

\begin{document}

\maketitle

这里是正文. 

\end{document}

    为了在文档中提示信息，需要使用\maketitle

    页面

    设置页边距：使用geometry宏包

    \usepackage{geometry}
\geometry{left=2.54cm, right=2.54cm, top=3.18cm, bottom=3.18cm}

    设置行间距：

    \linespread{1.5}

    页码

    默认的页码编码方式是阿拉伯数字，用户也可以自己设置为小写罗马数字

    \pagenumbering{roman}
    • aiph表示小写字母
    • Aiph表示大写字母
    • roman表示小写罗马数字
    • Roman表示大写罗马数字
    • arabic表示默认的阿拉伯数字

    如果要设置页码的话，可以用如下代码来设置页码从0开始：

    \setcounter{page}{0}

    正文

    在document环境中书写

    不需要加入空格来缩紧，因为文档默认会进行首行锁进。

    相邻的两行在编译时仍然会视为同一段，另起一行的方式是使用一行间隔（和markdown一致）。在正文部分，多余的空格和回车等等会被自动忽略

    另起一页的方式是：

    \newpage

    章节（分级标题）

    对于ctexart文件类型，章节可以用\section{}和\subsection{}命令来标记，例如：

    \documentclass[12pt, a4paper, oneside]{ctexart}
\usepackage{amsmath, amsthm, amssymb, graphicx}
\usepackage[bookmarks=true, colorlinks, citecolor=blue, linkcolor=black]{hyperref}

% 导言区

\title{我的第一个\LaTeX 文档}
\author{Dylaaan}
\date{\today}

\begin{document}

\maketitle

\section{一级标题}

\subsection{二级标题}

这里是正文. 

\subsection{二级标题}

这里是正文. 

\end{document}

    详解LaTeX分级标题

    七级标题

    分别为：part、chapter、section、subsection、subsubsection、paragraph、subparagraph。（ctexart类没有chapter结构）

    • 每级结构中都有一个带星号的版本，如\section*{}。带星号的标题命令仅仅起到输出标题的作用，而不会对标题进行编号，也不会把标题列入目录，也不会把标题写入页眉页脚。

    修改标题样式

    所有标题样式都在导言区用\ctexset命令进行设置

    设置标题编号前后的词语

    使用name参数进行设置

    name={前部分, 后部分}，其中,是编号占位符。

    如：

    \ctexset{
  % 修改 section。
  section={   
    name={,、}
  },
  % 修改 subsection。
  subsection={   
    name={,、}
  }
}

    效果为：

    可见 section 和 subsection 的编号后面换成了 、。

    设置标题编号

    使用number参数设置

    • \arabic设置为阿拉伯数字
    • \chinese设置为中文
    • \roman设置为小写罗马数字
    • \Roman设置为大写罗马数字
    • \alph设置为小写英文
    • \Alph设置为大写英文

    如：

    \ctexset{
  % 修改 section。
  section={   
    name={,、},
    number={\chinese{section}}
  },
  % 修改 subsection。
  subsection={   
    name={,、},
    number={\arabic{subsection}}
  }
}

    效果为：

    设置标题整体样式

    使用format参数设置，包括标题主题、编号以及编号前后的词语。

    format还可以设置标题的对齐方式。

    • \centering居中对齐
    • \raggedright左对齐
    • \hfill右对齐

    如：

    \ctexset{
    % 修改 section。
    section={   
        name={,、},
        number={\chinese{section}},
        format=\heiti\raggedright\zihao{-4} % 设置 section 标题为黑体、右对齐、小4号字
    },
    % 修改 subsection。
    subsection={   
        name={,、},
        number={\arabic{subsection}},
        format=\heiti\zihao{5} % 设置 subsection 标题为黑体、5号字
    }
}

    效果为：

    取消编号后的空白

    使用aftername参数控制编号与标题之间的距离

    如：

    \ctexset{
    % 修改 section。
    section={   
        name={,、},
        number={\chinese{section}},
        format=\heiti\raggedright\zihao{-4},
        aftername=hspace{0pt} % 设置距离为 0 磅
    },
    % 修改 subsection。
    subsection={   
        name={,、},
        number={\arabic{subsection}},
        format=\heiti\zihao{5},
        aftername=hspace{0pt} % 设置距离为 0 磅
    }
}

    效果为：

    设置标题与正文的距离

    使用beforeskip参数控制标题与上面正文的距离

    使用afterskip参数控制标题与下面正文的距离

    如：

    \ctexset{
    % 修改 section。
    section={   
        name={,、},
        number={\chinese{section}},
        format=\heiti\raggedright\zihao{-4},
        aftername=\hspace{0pt},
        beforeskip=1ex, % 修改距离
        afterskip=1ex  % 修改距离
    },
    % 修改 subsection。
    subsection={   
        name={,、},
        number={\arabic{subsection}},
        format=\heiti\zihao{5},
        aftername=\hspace{0pt},
        beforeskip=1ex, % 修改距离
        afterskip=1ex % 修改距离
    }
}

    效果为：

    其他重要的标题设置参数

    设置标题名字样式
    • format对标题整体进行样式设置，包括编号和标题主体。
    • 单独对编号设置，可以使用nameformat参数。
    设置编号数字样式
    • format对标题整体进行样式设置，包括编号和标题主体。
    • 单独对编号中的数字设置，可以使用numberformat参数。
    设置标题主体样式
    • format对标题整体进行样式设置，包括编号和标题主体。
    • 单独对标题主体进行设置，可以使用titleformat参数。

    目录

    有了章节的结构之后，在正文中使用\tableofcontents命令就可以在指定位置生成目录。

    通常带有目录的文件需要编译两次，因为需要先在目录中生成.toc文件，再据此生成目录

    \documentclass[12pt, a4paper, oneside]{ctexart}
\usepackage{amsmath, amsthm, amssymb, graphicx}
\usepackage[bookmarks=true, colorlinks, citecolor=blue, linkcolor=black]{hyperref}

% 导言区

\title{我的第一个\LaTeX 文档}
\author{Dylaaan}
\date{\today}

\begin{document}

\maketitle

\tableofcontents

\section{一级标题}

\subsection{二级标题}

这里是正文. 

\subsection{二级标题}

这里是正文. 

\end{document}

    图片

    插入图片需要使用graphicx宏包，建议使用如下方式：

    \begin{figure}[htbp]
    \centering
    \includegraphics[width=8cm]{图片.jpg}
    \caption{图片标题}
\end{figure}

    其中，[htbp]的作用是自动选择插入图片的最优位置，\centering设置让图片居中，[width=8cm]设置了图片的宽度为8cm，\caption{}用于设置图片的标题。

    表格

    LaTeX中表格的插入较为麻烦，可以直接使用Create LaTeX tables online – TablesGenerator.com来生成。建议使用如下方式：

    \begin{table}[htbp]
    \centering
    \caption{表格标题}
    \begin{tabular}{ccc}
        1 & 2 & 3 \\
        4 & 5 & 6 \\
        7 & 8 & 9
    \end{tabular}
\end{table}

    列表

    LaTeX中的列表环境包含无序列表itemize、有序列表enumerate和描述description，以enumerate为例，用法如下：

    \begin{enumerate}
    \item 这是第一点; 
    \item 这是第二点;
    \item 这是第三点. 
\end{enumerate}

    另外，也可以自定义\item的样式：

    \begin{enumerate}
    \item[(1)] 这是第一点; 
    \item[(2)] 这是第二点;
    \item[(3)] 这是第三点. 
\end{enumerate}

    定理环境

    定理环境需要使用amsthm宏包。

    首先在导言区加入：

    \newtheorem{theorem}{定理}[section]

    在这里：

    • {theorem}是环境的名称
    • {定理}表示该环境显示的名称是“定理”
    • [section]的作用是让theorem环境在每个section中单独标号

    在正文中，用如下方式加入一条定理：

    \begin{theorem}[定理名称]
    这里是定理的内容. 
\end{theorem}

    在这里：

    • [定理名称]不是必须的。

    另外，我们还可以建立新的环境，如果要让新的环境和theorem环境一起计数的话，可以用如下方式：

    \newtheorem{theorem}{定理}[section]
\newtheorem{definition}[theorem]{定义}
\newtheorem{lemma}[theorem]{引理}
\newtheorem{corollary}[theorem]{推论}
\newtheorem{example}[theorem]{例}
\newtheorem{proposition}[theorem]{命题}

    另外，定理的证明可以使用proof环境

    数学公式

    行内公式

    使用$...$输入，这通常被称为公式环境，例如：

    若$a>0$, $b>0$, 则$a+b>0$.
    • 公式环境通常使用特殊字体，并且默认为斜体。
    • 只要是公式，就必须放入公式环境中

    如果想要在行内公式中展现出行间公式的效果，需要在前边加入\displaystyle：

    设$\displaystyle\lim_{n\to\infty}x_n=x$.

    行间公式

    使用$$...$$输入：（与markdown一致）

    若$a>0$, $b>0$, 则
$$
a+b>0.
$$
    • 行间公式也是正文的一部分，需要与正文连贯，并且加入标点符号

    上下标

    上标：用^输入，如：a^n

    下标：用_输入，如：a_1

    • 上下标只会读取一个字符，若字符较多的话，需要用大括号括起来。如：a_{max}

    分式

    用\dfrac{}{}输入，如：\dfrac{a}{b}，即b分之a。

    在行间，分子，分母，指数上输入较小的分式，则需要改用\frac{}{}，如：a^\frac{1}{n}。

    括号

    直接用(...)输入

    如果括号内的内容较大，需要改用\left(...\right)，如：

    \left(1+\dfrac{1}{n}\right)^n

    在中间需要隔开时，可以用\left(...\middle|...\right)。

    另外，输入花括号时需要用\{...\}，\起到转义的作用。

    加粗

    建议加载bm宏包，使用\bm{}来加粗，这可以保留公式的斜体

    大括号

    使用cases环境，可以用于分段函数或方程组，例如：

    $$
f(x)=\begin{cases}
    x, & x>0, \\
    -x, & x\leq 0.
\end{cases}
