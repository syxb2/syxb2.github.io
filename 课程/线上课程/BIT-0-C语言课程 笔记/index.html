    ```c int main() { printf("%d\n", 100); //打印整数100 //%d - 指定格式：打印一个整数 //%lf - 打印一个浮点数

sizeof的单位是：字节（byte）

    注：
    计算机中常见的单位：（由大到小）

    bit - 比特位 - 最小
        //计算机中识别二进制

    byte - 字节

    1 kb = 1024 byte
    1 mb = 1024 kb
    1 gb = 1024 mb
    1 tb = 1024 gb
    1 pb = 1024 tb

#### 为什么单是整型就分为int、short、long、long long呢？

每种数据类型所占用空间大小固定，其所占空间大小决定了它储存能力的上限

故（对于同种类型（如整型））要合理选用不同数据类型以使空间利用率最高

    第五节.变量的作用域和生命周期

    作用域

    通俗来说就是这个变量在哪里可以使用哪里就是它的作用域

    如int函数的作用域就是其所在的大括号内 

    #include <stdio.h>

int main() {

    {
        int a = 0;
    }
    printf("%d\n", a);

    return 0;
}

    以上代码就是错误的，会提示a变量没有声明

    上面的a就只能在int a = 0所在的大括号内使用

    局部变量

    局部变量的作用域就是变量所在的局部范围

    全局变量

    全局变量的作用域是整个代码区

    #include <stdio.h>

int a = 0; // 声明全局变量a 

// 执行性的代码必须写在主函数的里面

int main() {

    printf("%d\n", a);
    return 0;
}

    同一个工程中跨源文件使用全局变量：需要在没有声明变量的文件中声明一下extern int a

    声明周期

    变量的创建和销毁之间的时间段就是变量的生命周期

    • 局部变量的生命周期：从进入局部范围开始到出局部范围结束
    #include <stdio.h>

int main() {
    {
        int a = 0;
        printf("%d\n", a);
    }

}

    如这里就是在int a = 0;开始从出了其所在大括号范围结束

    • 全局变量的生命周期：整个程序

    第六节.常量

    分类

    1. 字面常量

    如3.14，8，9

    1. const修饰的常变量

    其本身具有常属性，即不能被改变。但本质还是一个变量。

    例如：

    const int a = 10;
int arr[a] = {0};
// 这种写法是不被允许的

    1. define定义的标识符常量

    #define MAX 10000
int n = MAX;
printf("%d\n", n)
// 程序输出为10000
    1. 枚举常量

    可以一一列举的常量

    例：

    enum Sex
{
    // 这种枚举类型的变量的未来可能取值
    // 枚举常量
    MALE,
    FEMALE,
    SECRET
};

int main()
{
    enum Sex s = MALE;

    printf("%d\n", MALE);
    printf("%d\n", FEMALE);
    printf("%d\n", SECRET);

    return 0;
}
// 程序的输出值为：
// 0
// 1
// 2
// 枚举常量是常量，它的值是默认从零开始的。

    第七节.字符串和转义字符

    字符串

    字符串就是一串字符 - 用双引号括起来的一串字符

    int main()
{
    // 字符数组 - 数组是一组同类型的元素
    // 字符串在结尾的位置隐藏了一个\0的字符
    // \0是字符串的结束标志，但不会算做字符串的内容
    char arr1[] = "abc";
    char arr2[] = {'a', 'b', 'c'}
    // 字符串用双引号包围，单个字符用单引号包围

    // 打印字符串
    printf("%s\n", arr1); // %s - 字符串
    printf("%s\n", arr2);
    // arr1中有四个字符a, b, c, \0
    // arr2中有三个字符a, b, c

    // 求字符串长度
    #include <string.h> // strlen所在的头文件 // 需要放到main函数外面

    int len = strlen("abc");
    printf("%d\n", len);
    // 输出为3 - 在计算字符串长度时不包括\0

    return 0;
}

    打印时遇到\0会停下来，如果没有\0则会继续向后打印，但abc后面的内存里不知道是什么，所以会出现乱码

    转义字符

    转变了它原来的意思，如：\n - 不是一个\一个n两个字符，而是换行的意思

    两个重要的转义字符：

    1. \ddd - 将三位八进制数字转义为十进制数。如\130等价于88
    2. \xdd - 将两位十六进制数字转义为十进制数。如\x30等价于48

    注意：

    \ddd或\xdd只算一个转义字符

    指针

    指针就是地址

    如使用指针变量pa时就是在使用pa里存的地址

    内存

    1. 内存是怎么编号的？

    32/64位计算机 - 32/64根地址线 - 物理线 - 通电表示1，不通电表示0

    电信号转换成数字信息：1和0组成的二进制序列

    内存的编号就是二进制序列，这个编号就称为这个内存单元的地址，对于32位计算机来说，每个二进制序列由32个数字组成（二进制），所以32位计算机的内存编号有2^32种可能。

    1. 一个内存单元是多大空间？

    最终，一个内存单元是1byte（1字节）

    指针变量（*）

    int main()
{
    int a = 10; // a在内存中要分配空间 - 4个字节 - a的地址是四个地址的第一个
    printf("%p\n", &a); // $p专门用来打印地址

    int * pa = &a; // 把a的地址存入pa
    // pa是用来存放地址的，在c语言中，我们就称pa为指针变量
    // * 说明 pa是指针变量
    // int 说明 pa执行的对象是int类型的

    // 如果：
    char ch = 'a';
    // 则要写
    char * pc = &ch; // 表示pc的执行对象是char类型的

    return 0;
}

    指针的作用

    int main()
{
    int a = 10;

    int * pa = &a;

    *pa = 20; // * 是 解引用操作符 - *pa就是通过pa内存的地址找到此地址存的值，即a
    // 相当于把a的值间接改为了20
}

    int *p = num - 相当于p这个指针指向了num

    *p = 20 - 把指针指向的内存单元内的值改为20

    指针变量的大小

    指针的大小是相同的：32位机器 - 4byte；64位机器 - 8byte

    为什么呢？

    指针是用来存放地址的

    指针需要多大空间取决于地址的存储需要多大空间

    指针的大小：

    指针的大小跟地址线的多少有关，而地址线又与计算机位数有关

    以32为计算机为例：一个地址由32个二进制数字构成，所以需要32bit的内存，即4byte

    x86 - 32位机器；x64 - 64位机器
